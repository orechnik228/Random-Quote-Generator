# Random-Quote-Generator
Автор: [Илиади Станислав]

Описание
«Random Quote Generator» — это простое GUI-приложение на Python, которое позволяет пользователям получать случайные мудрые высказывания. Приложение сохраняет историю всех сгенерированных цитат в локальный JSON-файл и предоставляет инструменты для фильтрации истории по автору или теме.

Как использовать
Запуск: Убедитесь, что у вас установлен Python, и запустите файл Random Quote Generator.py.

Генерация: Нажмите кнопку «Сгенерировать цитату», чтобы увидеть случайное высказывание на главном экране. Цитата будет автоматически добавлена в таблицу истории ниже.

Фильтрация:

Введите имя автора или название темы в поле «Фильтр (Автор/Тема)».

Нажмите кнопку «Фильтровать», чтобы отобразить в списке только соответствующие записи.

Примеры использования
Генерация: Нажмите кнопку «Сгенерировать цитату» — в центральной части окна появится новая цитата, а в таблице под ней отобразится история всех ваших запросов.

Тест фильтрации: Введите в поле фильтра слово Наука и нажмите «Фильтровать» — таблица покажет только те цитаты, которые относятся к данной теме.

Технические детали
Хранение данных: История цитат автоматически сохраняется в файл history.json.

Контроль версий: В проекте используется Git. Файл .gitignore настроен на игнорирование локального файла истории history.json и кэша Python, чтобы поддерживать чистоту вашего репозитория.
